# Lumen Crypto Exchange

A standalone premium frontend skeleton for a cryptocurrency exchange with trading pairs, a candlestick-style chart, order book, trade form, open orders, and portfolio management.

## Files

- `index.html` contains the complete inline HTML, CSS, and JavaScript implementation.
- `DESIGN_BOOK.md` documents the visual system and interaction choices.

## Usage

Open `index.html` in a browser. The template has no build step and no runtime dependencies beyond Google Fonts.

## Included Sections

- Top navigation with search and notification badge
- BTC/USDT trading pair selector with 24h market metrics
- CSS candlestick-style price chart with timeframe toggles
- Split order book with bids and asks
- Buy and sell trade form with order type, amount, price, and total
- Open orders table with cancel controls
- Portfolio overview with allocation bars
- Footer credit
