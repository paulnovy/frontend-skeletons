# Cortex AI Chat

A standalone premium frontend skeleton for an AI chatbot UI with conversation history, model selection, prompt templates, and settings controls.

## Files

- `index.html` contains inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system and component intent.
- `README.md` explains usage and structure.

## Usage

Open `index.html` in a browser. The page has no runtime dependency beyond Google Fonts.

## Features

- Sticky top navigation with model dropdown.
- Desktop history sidebar and mobile hamburger drawer.
- AI and user message bubbles with timestamps and copy buttons.
- Prompt composer with character count and keyboard shortcut hint.
- Model selector cards with provider, speed, and token pricing.
- Prompt template grid with insertion behavior.
- Settings panel with temperature, max tokens, system prompt, and toggles.
- Accessible focus states, semantic regions, and reduced-motion support.
