# Cortex AI Chat Design Book

## Product
Cortex AI Chat is a premium AI chat interface for teams that need conversation history, model selection, prompt templates, and practical generation controls in one focused workspace.

## Typography
- Headings use Manrope from Google Fonts.
- Interface text uses Inter from Google Fonts.
- Letter spacing remains neutral for a calm professional reading rhythm.

## Color System
- Cloud #F6F8FC is the page background and soft panel fill.
- Ink #19212B is the core text color and dark action surface.
- Indigo #4F46E5 is the primary action and selected state.
- Cyan #06B6D4 is used for live indicators and secondary controls.
- Lime #A3E635 communicates healthy connection status.
- Coral #FB7185 supports alert and close-control emphasis.

## Layout
The desktop layout uses a persistent history sidebar with the chat workspace as the primary region. Mobile moves the history into a keyboard-navigable slide-in drawer with a backdrop and Escape support.

## Components
- Top navigation with section links and model dropdown.
- Conversation history with title, date, preview, and active state.
- Chat messages with avatars, timestamps, user alignment, and assistant copy actions.
- Composer with attachment action, character count, send button, and shortcut hint.
- Model cards showing provider, speed, and token cost.
- Prompt templates with category tags and direct insertion behavior.
- Settings panel for temperature, max tokens, system prompt, streaming, and memory.

## Accessibility
The page includes semantic landmarks, skip navigation, focus-visible styles, aria labels, drawer state management, live message region, keyboard Escape handling, and reduced-motion support.
