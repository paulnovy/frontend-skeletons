# Zen Wellness Retreat

Standalone premium frontend skeleton for a wellness retreat booking business.

## Included

- `index.html` with inline CSS and JavaScript
- Google Fonts CDN loading for Manrope and Inter
- Responsive desktop, tablet, and mobile layouts
- Keyboard-accessible hamburger drawer
- Program, practitioner, meditation, booking, testimonial, and footer sections

## Usage

Open `index.html` directly in a browser or serve the repository with the existing frontend-skeletons tooling.
