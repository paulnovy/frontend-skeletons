# Spark Creator Economy

Premium standalone frontend skeleton for a creator monetization platform with audience analytics, content scheduling, sponsorship workflow, and revenue tracking.

## Files

- `index.html` — single-file HTML, inline CSS, and inline JavaScript.
- `DESIGN_BOOK.md` — design rationale, palette usage, layout, and accessibility notes.
- `README.md` — implementation summary.

## Requirements Covered

- Manrope headings and Inter UI text loaded from Google Fonts CDN.
- Cloud, ink, indigo, cyan, lime, and coral palette with tints through opacity.
- Top nav, KPI strip, content calendar, audience analytics, revenue tracker, top content table, sponsorship inbox, and visible footer credit.
- Mobile hamburger drawer with keyboard support, Escape close, focus-visible styling, and reduced motion handling.
- No external dependencies beyond Google Fonts.

## Preview

Open `index.html` in a browser or serve the repository with Vite from the `frontend-skeletons` workspace.
