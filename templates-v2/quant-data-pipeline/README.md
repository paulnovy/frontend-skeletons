# Quant Data Pipeline

Premium frontend skeleton for a data pipeline operations dashboard with DAG visualization, job monitoring, data quality scoring, alerts, and run history.

## Structure

- `index.html` contains the standalone HTML, CSS, and JavaScript.
- Google Fonts loads Manrope for headings and Inter for interface text.
- No runtime dependencies are required beyond the font CDN.

## Preview

Open `index.html` in a browser to review the dashboard. The mobile drawer, refresh control, alert actions, table, DAG, KPI strip, and quality cards are included as production-ready UI scaffolding.
