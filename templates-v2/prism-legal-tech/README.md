# Prism Legal Tech

Premium frontend skeleton for legal practice management with case tracking, document assembly, calendar planning, billing integration, and client management.

## Files
- `index.html` contains the standalone HTML, CSS, and JavaScript.
- `DESIGN_BOOK.md` documents the design system, layout, and accessibility decisions.

## Usage
Open `index.html` in a browser. The file is standalone and only loads Manrope and Inter from the Google Fonts CDN.

## Included Sections
- Top navigation with brand, links, search, notification badge, and mobile drawer.
- KPI strip for active cases, upcoming deadlines, unbilled hours, and outstanding invoices.
- Case tracking cards for Civil, Criminal, Corporate, and Family matters.
- Document assembly template table with generate actions.
- Monthly calendar with event dots and upcoming deadline sidebar.
- Billing integration table with invoice actions.
- Client management cards with active cases and billed totals.
- Footer credit: Prism Legal Tech — designed by Meridian Studio.

## Accessibility
The mobile drawer supports keyboard focus, Escape close, focus return, and visible focus states. The layout uses semantic landmarks and honors reduced-motion preferences.
