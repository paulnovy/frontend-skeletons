# Horizon Investor Relations

Premium standalone investor relations frontend skeleton for a fintech corporate IR page.

## Included

- `index.html` with inline CSS and JavaScript
- Google Fonts loading for Manrope and Inter
- Sticky top navigation with mobile hamburger drawer
- KPI strip for stock price, market cap, 52-week range, and dividend yield
- Financial report cards
- CSS-only six-month stock chart
- Key metrics table
- Governance board cards
- Upcoming investor events
- Shareholder resource links
- Footer credit and legal disclaimer

## Usage

Open `index.html` in a browser. The template is dependency-free except for the Google Fonts CDN request.

## Customization

Replace the sample issuer data, board members, report dates, stock metrics, and resource links with production investor relations content. Keep the defined palette intact for brand consistency.
