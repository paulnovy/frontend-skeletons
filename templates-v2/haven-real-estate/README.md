# Haven Real Estate

Standalone premium frontend skeleton for a real estate marketplace with property search, virtual tour preview, neighborhood insights, mortgage calculator, and agent spotlight.

## Files

- `index.html` contains the complete standalone experience with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system, sections, responsive behavior, and accessibility approach.
- `README.md` records usage and verification notes.

## Usage

Open `index.html` directly in a browser. The page has no build step and no runtime dependency beyond the Google Fonts CDN request for Manrope and Inter.

## Verification

Run the required HTML parser, line count, font, palette, and forbidden-text checks after creation. Responsive verification should be performed with Playwright at desktop `1440`, tablet `768`, and mobile `390`, including a mobile screenshot for inspection.
