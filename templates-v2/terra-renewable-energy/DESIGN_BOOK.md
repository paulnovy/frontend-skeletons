# Terra Renewable Energy Design Book

## Concept
Terra Renewable Energy is a standalone renewable operations dashboard for monitoring solar, wind, storage, grid balance, carbon offsets, and production forecasts. The interface is dense enough for operators while staying clean for executive review.

## Typography
- Headings: Manrope, loaded from Google Fonts.
- Interface text: Inter, loaded from Google Fonts.

## Palette
- Cloud: #F6F8FC for the application background.
- Ink: #19212B for primary text and high-contrast panels.
- Indigo: #4F46E5 for grid and storage emphasis.
- Cyan: #06B6D4 for wind, live status, and network energy flow.
- Lime: #A3E635 for solar, active status, and positive renewable performance.
- Coral: #FB7185 for storage dispatch and carbon accents.

## Layout
The page uses a sticky top navigation, a responsive hero with a live network map, KPI strip, stacked CSS production chart, site monitoring cards, grid management gauges, carbon tracking panels, and hourly forecast cards. Mobile navigation collapses into a hamburger drawer.

## Interaction
The dashboard is static and standalone, with inline JavaScript only for opening and closing the mobile drawer. All visuals are CSS, HTML, and inline SVG icons.
