# Signal IoT Fleet

Signal IoT Fleet is a standalone IoT fleet monitoring dashboard skeleton.

It includes:
- anomaly-first incident banner
- dense fleet health summary
- operational SVG asset map
- device health cards
- alert queue
- telemetry charts
- maintenance prediction scores
- mobile field technician view

Fonts: IBM Plex Sans and IBM Plex Mono.

Palette: #0A0A0A, #374151, #06B6D4, #F97316, #22C55E, #F3F4F6.
