# Aurora Robotics Control

A standalone robotics fleet management template for monitoring robot status, assigning tasks, scheduling maintenance, and reviewing telemetry.

## Files

- `index.html` contains the complete responsive interface with inline CSS and JavaScript.
- `DESIGN_BOOK.md` captures design tokens, layout notes, and interaction guidance.

## Design Notes

The template uses Manrope for headings, Inter for UI text, and the requested Cloud, ink, indigo, cyan, lime, and coral palette. It includes desktop navigation, a hamburger drawer for mobile, KPI metrics, fleet cards, a task Kanban board, maintenance scheduling, telemetry charts, and the required Studio footer credit.
