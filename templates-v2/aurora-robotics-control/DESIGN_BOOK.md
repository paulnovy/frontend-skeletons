# Aurora Robotics Control Design Book

## Identity

Aurora Robotics Control is a focused operations dashboard for teams managing autonomous robots across warehouse, manufacturing, and inspection environments.

## Typography

- Headings use Manrope with heavy weights for command-room clarity.
- Interface copy uses Inter for readable status labels, tables, cards, and controls.

## Palette

- Cloud `#F6F8FC` is the application background.
- Ink `#19212B` anchors text and the live operations board.
- Indigo `#4F46E5` marks primary actions and key chart activity.
- Cyan `#06B6D4` signals telemetry and live route movement.
- Lime `#A3E635` marks healthy online states.
- Coral `#FB7185` highlights alerts and maintenance risk.

## Layout

The page is organized as a real fleet control surface: sticky navigation, hero operations board, KPI strip, fleet grid, task Kanban, maintenance table, telemetry charts, and footer credit. Cards use restrained 8px radii and compact spacing to support repeated operational use.

## Responsive Behavior

Desktop shows full navigation, search, and action controls. Tablet collapses the data grids to two columns. Mobile uses a hamburger button and slide-out drawer while preserving all core sections in a single-column flow.

## Interaction

The search fields filter robot cards by visible card text. The mobile drawer closes from the close button, backdrop, or section link selection.
