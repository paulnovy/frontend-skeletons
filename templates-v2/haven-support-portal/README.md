# Haven Support Portal

Premium standalone frontend skeleton for a customer support portal with ticket management, knowledge base cards, live chat, ticket detail, and satisfaction reporting.

## Files
- `index.html` — single-file HTML/CSS/JS implementation
- `DESIGN_BOOK.md` — design rationale and system notes
- `README.md` — usage overview

## Usage
Open `index.html` in a browser. No build step is required. The only external request is Google Fonts for Manrope and Inter.

## Included Sections
- Top navigation with search, notifications, and mobile drawer
- KPI strip for service metrics
- Sortable ticket table
- Knowledge base search and article cards
- Live chat panel with canned responses
- Expandable ticket detail view
- CSS donut chart for customer satisfaction
- Footer credit
