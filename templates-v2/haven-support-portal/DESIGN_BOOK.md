# Haven Support Portal Design Book

## Concept
Haven Support Portal is a premium support operations skeleton for ticket triage, self-service content, live chat, and satisfaction monitoring. The interface is calm, dense, and built for repeated daily work.

## Typography
Headings use Manrope for a confident operational tone. Interface text uses Inter for legibility across tables, controls, and compact panels. Both families load from Google Fonts in `index.html`.

## Color System
The palette is intentionally narrow: Cloud `#F6F8FC`, Ink `#19212B`, Indigo `#4F46E5`, Cyan `#06B6D4`, Lime `#A3E635`, and Coral `#FB7185`, with opacity-based tints derived from those colors.

## Layout
The top navigation anchors the workspace with global search and notification state. The main surface starts with operational context, then moves into KPI cards, ticket triage, knowledge base health, live chat, ticket detail, and CSAT reporting.

## Interaction Notes
The mobile navigation uses a keyboard-accessible slide-in drawer. Ticket table headers sort rows in place. Detail cards use native disclosure controls for accessible expansion.

## Accessibility
Semantic landmarks, visible focus states, reduced-motion support, labeled controls, accessible drawer state, and table captions are included in the standalone file.
