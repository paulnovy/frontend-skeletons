# Roam Travel Booking

Premium travel booking skeleton that connects immersive destination discovery with practical booking confidence: flexible dates, map/list results, curated trips, total price clarity, social proof, and insurance upsell.

## Includes

- Immersive destination hero
- Search module with destination, dates, and travelers
- Flexible date cards
- Map/list hybrid results
- Curated trip cards
- Social proof section
- Travel insurance upsell
- Sticky mobile search summary

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Plus Jakarta Sans and Source Serif 4. Palette follows the brief: sand, ocean, sunset, palm, night, and cloud.
