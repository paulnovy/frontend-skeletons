# Liquid Identity Onboarding

## Category
Premium frontend skeleton · Futuristic identity verification onboarding

## Description
Liquid Identity Onboarding is a standalone HTML skeleton for a polished four-step verification flow: Enter Info, Verify Identity, Review, and Confirm. It uses liquid-inspired transitions, biometric accents, realistic validation copy, and a mobile-first responsive layout.

## How to View
Open `index.html` directly in a browser. The file contains inline CSS and JavaScript, with Google Fonts loaded from the CDN.

## Features
- Standalone single-file frontend with inline CSS and JavaScript.
- Four-step identity verification form with animated state transitions.
- Mobile hamburger menu and responsive desktop navigation.
- Realistic legal identity fields, verification statuses, consent copy, and error messages.
- Premium color system using deep teal, liquid cyan, pearl, biometric violet, and coral.
- Semantic HTML5 structure, focus states, keyboard support, and reduced-motion fallback.
- Footer text: “Built with Frontend Skeletons”.
