# Beacon Waitlist

A standalone premium frontend skeleton for a startup pre-launch waitlist page with referral sharing, countdown, social proof, and FAQ interactions.

## Files

- `index.html`: Single-file implementation with inline CSS and JavaScript.
- `DESIGN_BOOK.md`: Design rationale, color system, typography, layout, and accessibility notes.
- `README.md`: Usage and verification notes.

## Features

- Sticky navigation with mobile hamburger drawer
- Hero email capture form
- Live countdown timer
- Three feature preview cards
- Referral link display with copy, Twitter, and email actions
- Referral progress bar and reward tiers
- Social proof logo strip and testimonial quotes
- Five-item FAQ accordion
- Visible footer credit

## Constraints

- Uses Manrope and Inter from Google Fonts.
- Uses only the required color system and opacity-based tints.
- No external dependencies beyond Google Fonts.
- Designed as a single static HTML page.

## Run

Open `index.html` in a browser. No build step is required.

## Verify

Run the project verification command from the task to confirm HTML parsing, line count, font names, color tokens, and forbidden text checks.
