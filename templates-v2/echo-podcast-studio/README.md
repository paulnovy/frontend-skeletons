# Echo Podcast Studio

Premium frontend skeleton for a podcast management platform with episode scheduling, analytics, listener engagement, and distribution monitoring.

## Files

- `index.html` contains the standalone dashboard with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system, layout decisions, and accessibility notes.

## Requirements Met

- Manrope headings and Inter UI text from Google Fonts.
- Required color system with cloud, ink, indigo, cyan, lime, and coral.
- Responsive top navigation with mobile hamburger drawer.
- KPI strip, episode table, analytics chart, scheduler, engagement cards, distribution cards, and visible footer credit.
- No external runtime dependencies beyond Google Fonts.

Open `index.html` in a browser to review the skeleton.
