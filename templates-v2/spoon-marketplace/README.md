# Spoon Marketplace

Search-first food delivery marketplace skeleton for local restaurant ordering, built around the signals customers actually need before checkout: cuisine, ETA, rating, delivery fee, dietary tags, and loyalty value.

## Includes

- Location and meal search hero
- Cuisine chip filters
- Featured restaurant cards
- Delivery ETA cards
- Weekly editorial picks
- Loyalty banner with points preview
- Sticky mobile cart
- Responsive drawer navigation

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Bricolage Grotesque and Inter. Palette follows the brief: warm cream, tomato, basil, saffron, charcoal, and light mint.
