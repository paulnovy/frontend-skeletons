# Wave Social Platform

Premium standalone frontend skeleton for a social networking feed with profile preview, content sharing, notifications, trends, suggested follows, and messaging.

## Files
- `index.html` contains the complete inline HTML, CSS, and JavaScript.
- `DESIGN_BOOK.md` documents the visual system and interaction model.

## Usage
Open `index.html` in a browser. No build step is required.

## Requirements Covered
- Manrope headings and Inter UI typography from Google Fonts.
- Required palette with Cloud, ink, indigo, cyan, lime, and coral.
- Mobile hamburger menu with keyboard-accessible drawer controls.
- Messaging panel with conversation list, chat bubbles, and input.
- Notification dropdown with accessible state handling.
- Visible footer credit.
