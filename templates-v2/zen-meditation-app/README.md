# Zen Meditation App

A standalone premium frontend skeleton for a meditation and mindfulness app.

## Includes
- Responsive top navigation with mobile hamburger drawer
- Daily greeting hero with animated breathing circle
- Guided session library
- Breathing exercise controls
- Weekly progress tracking and mood comparison
- Soundscape mixer with sliders and play states
- Achievement badge grid

## Run
Open `index.html` in a browser. The file is standalone and only loads Manrope and Inter from Google Fonts.
