# Prism AI Product Landing

## Category
AI SaaS Homepage / Transparent AI Decision Platform

## Description
Prism AI Product Landing is a premium standalone frontend skeleton for an AI SaaS product that emphasizes trustworthy, inspectable AI. It includes confidence scores, model/version labels, audit trail previews, governance messaging, human override controls, use-case tabs, pricing, testimonials, FAQ, and a mobile sticky CTA.

## Files
- `index.html`: Standalone HTML file with inline CSS and JavaScript.
- `DESIGN_BOOK.md`: Full design system, token list, layout notes, accessibility notes, and content rules.
- `README.md`: Project overview and usage notes.

## How to View
Open `index.html` directly in a browser, or serve this folder locally:

```bash
cd /home/marian/clawd/frontend-skeletons/templates-v2/prism-ai-product-landing
python3 -m http.server 3010
```

Then visit:

```text
http://localhost:3010/
```

## Key Features
- Large split hero with left product copy and right animated AI workflow diagram.
- Google Fonts CDN: Sora headings, Manrope body, JetBrains Mono model labels.
- Required palette: Porcelain, ink, ultraviolet, soft coral, mist blue, and charcoal.
- Trust strip, bento feature grid, transparent decision logic section, use-case tabs, pricing preview, testimonials, and FAQ.
- Realistic AI SaaS content with model names, confidence percentages, audit IDs, policy checks, and reviewer states.
- Mobile hamburger menu and keyboard-friendly tab navigation.
- Mobile workflow stepper behavior and sticky CTA after first scroll.
- Visible focus states, semantic HTML5, accessible contrast, and reduced-motion fallback.
- Footer credit: "Built with Frontend Skeletons".
