# Forge Venture Capital

A standalone VC fund dashboard skeleton for portfolio monitoring, deal flow review, investment tracking, and fund performance reporting.

## Files

- `index.html` contains inline CSS, inline JavaScript, and Google Fonts CDN links.
- `DESIGN_BOOK.md` documents the visual system and layout rationale.
- `README.md` describes the template purpose and structure.

## Features

- Sticky top navigation with search and Add Deal action.
- Mobile hamburger menu with slide-out drawer.
- KPI strip for fund size, portfolio count, total invested, and IRR.
- Portfolio company cards with sector and stage badges.
- Kanban-style deal flow pipeline.
- Investment tracking table.
- CSS-only NAV chart with DPI, TVPI, and IRR cards.

## Usage

Open `index.html` in a browser. No build step is required.
