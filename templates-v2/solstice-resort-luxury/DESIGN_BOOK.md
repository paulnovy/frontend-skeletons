# Solstice Resort Luxury - Design Book

## Project Name
Solstice Resort Luxury

## Product Type
Luxury hotel and private resort website.

## Intended Audience
Luxury resort operators, boutique hospitality groups, travel agencies, concierge teams, and design studios building premium booking experiences.

## User Problem
Luxury hotel sites often bury booking behind cinematic presentation, or make booking feel too transactional. Solstice keeps reservation controls obvious without breaking the premium atmosphere.

## UX Concept
The page combines a cinematic resort hero with a clear booking bar, room category cards, experience timeline, dining story, gallery, reviews, and location map.

## Layout Structure
Sticky nav, full-bleed hero, booking bar overlay, suite card grid, day timeline, dining section, cinematic gallery, review band, map block, mobile sticky booking drawer, and footer.

## Typography System
Playfair Display creates elegant luxury headlines. Manrope handles booking UI, navigation, labels, cards, and transactional clarity.

## Color System
Ivory `#FAF3E3`, espresso `#2C1A12`, olive `#556B2F`, gold `#C6A15B`, seafoam `#B7D7C9`, and charcoal `#1F1F1F`.

## Component System
Core components include mobile drawer, booking field group, room cards, timeline cards, dining feature, menu cards, gallery cards, review cards, map pins, and sticky mobile booking CTA.

## Interaction Rules
Booking fields remain simple and legible. Drawer state updates `aria-expanded` and `aria-hidden`. Mobile hides the wide booking bar and exposes a sticky booking control.

## Responsive Behavior
Desktop emphasizes cinematic scale and overlay booking. Tablet stacks booking and content grids. Mobile converts gallery into vertical story cards and booking into a sticky bottom drawer trigger.

## Accessibility Notes
Semantic landmarks, skip link, labeled booking controls, visible focus states, readable contrast, descriptive links/buttons, accessible drawer state, and reduced-motion fallback are included.

## Performance Notes
Standalone HTML with inline CSS, minimal JavaScript, CSS-generated visual blocks, stable card dimensions, and no heavy dependencies.

## Commercial Use Cases
Luxury resort homepage, boutique hotel booking funnel, hospitality campaign page, room category showcase, destination retreat website, and concierge-led booking prototype.
