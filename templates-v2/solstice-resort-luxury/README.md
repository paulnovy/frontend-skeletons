# Solstice Resort Luxury

Luxury resort website skeleton with cinematic storytelling and booking controls that stay clear: hero, booking bar, room categories, experience timeline, dining, gallery, reviews, and location map.

## Includes

- Cinematic resort hero
- Booking bar overlay
- Suite category cards
- Experience timeline
- Dining section
- Responsive gallery
- Reviews and location map
- Sticky mobile booking control

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Playfair Display and Manrope. Palette follows the brief: ivory, espresso, olive, gold, seafoam, and charcoal.
