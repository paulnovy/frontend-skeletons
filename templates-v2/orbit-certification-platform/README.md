# Orbit Certification Platform

Professional certification platform skeleton with exam management, study materials, progress tracking, and certificate generation surfaces.

## Files

- `index.html` — standalone HTML with inline CSS and JavaScript.
- `DESIGN_BOOK.md` — visual system notes and component rationale.
- `README.md` — project overview.

## Features

- Sticky top nav with catalog search and mobile hamburger drawer.
- KPI strip for earned certificates, study hours, pass rate, and streak.
- Certification catalog with progress bars and level badges.
- Weekly study planner.
- Interactive exam simulator answer selection and countdown timer.
- Certificate cards with dates, expiry, and download actions.
- Progress dashboard with mastery bars, readiness chart, and activity feed.

## Usage

Open `index.html` in a browser. The skeleton is standalone and uses Google Fonts as its only external dependency.
