# Nexus SaaS Landing

Standalone premium frontend skeleton for a SaaS product landing page.

## Files

- `index.html` - standalone HTML with inline CSS and JavaScript
- `DESIGN_BOOK.md` - design tokens, layout guidance, UX rules, and content model
- `README.md` - usage and customization notes

## Features

- Sticky top navigation with search, CTA, and keyboard-friendly mobile drawer
- Hero section with email capture, proof points, social proof logos, and product dashboard preview
- Six-card feature grid, three-step visual workflow, pricing tiers, testimonials, FAQ accordion, final CTA, and footer
- Manrope headings and Inter UI loaded from Google Fonts
- Required cloud, ink, indigo, cyan, lime, and coral palette with opacity-based tints
- Semantic landmarks, visible focus states, Escape-to-close drawer behavior, and reduced-motion support

## Usage

Open `index.html` directly in a browser. The page has no runtime dependencies beyond the Google Fonts CDN and includes all styling and behavior inline.

## Customization

- Edit CSS custom properties in `:root` to adjust color intensity, shadows, spacing, and type.
- Replace hero metrics and signal feed rows with product-specific customer data.
- Update pricing tier copy and feature lists to match the SaaS offer.
- Swap social proof names, testimonials, and footer studio credit for the target brand.
