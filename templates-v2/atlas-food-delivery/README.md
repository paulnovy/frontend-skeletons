# Atlas Food Delivery

A standalone HTML frontend skeleton for a food delivery platform with restaurant listings, order tracking, menu management, driver dispatch, and analytics.

## Files
- `index.html` — single-page app with inline CSS and JavaScript.
- `DESIGN_BOOK.md` — design system notes, palette, typography, and layout rationale.
- `README.md` — project overview.

## Usage
Open `index.html` in a browser. The template uses Google Fonts CDN for Manrope and Inter, with all other styling and behavior embedded in the file.

## Included Sections
- Top navigation with search, cart count, and mobile drawer.
- Restaurant grid with cuisine badges, star ratings, delivery time, minimum order, and featured tags.
- Active order tracking with timeline states, ETA, and driver information.
- Menu builder with category tabs, customizations, and add-to-cart behavior.
- Driver dispatch board with courier statuses and current deliveries.
- Analytics cards, popular items, and a revenue chart.
