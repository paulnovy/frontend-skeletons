# Arch Agency Site

A standalone premium frontend skeleton for a digital agency website.

## Includes
- Single-file `index.html` with inline CSS and JavaScript
- Google Fonts CDN for Manrope and Inter
- Mobile hamburger drawer
- Service cards, case studies, team profiles, process flow, contact form, and footer

## Usage
Open `index.html` in a browser or serve the directory with any static file server.
