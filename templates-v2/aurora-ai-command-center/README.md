# Aurora AI Command Center

Standalone premium frontend skeleton for an AI operations command center covering real-time model monitoring, prompt management, inference analytics, cost allocation, and recent activity.

## Files

- `index.html` contains the complete page, inline CSS, and inline JavaScript.
- `DESIGN_BOOK.md` documents palette, typography, layout, and accessibility decisions.
- `README.md` records usage and verification notes.

## Usage

Open `index.html` directly in a browser. The page has no build step and no runtime dependency beyond the Google Fonts CDN request for Manrope and Inter.

## Verification

The required parser, line count, palette, typography, and forbidden-text checks were run after creation. Responsive verification was performed with Playwright at desktop `1440`, tablet `768`, and mobile `390`, including a mobile screenshot for visual inspection.
