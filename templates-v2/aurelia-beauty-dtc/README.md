# Aurelia Beauty DTC

Premium beauty brand e-commerce skeleton for skincare teams that want product education, routine discovery, bundle merchandising, verified reviews, and refill subscription conversion in one lightweight page.

## Includes

- Product-led hero with CSS product illustration
- Ingredient cards with practical benefit copy
- Accessible before/after comparison panel
- Guided routine builder with mobile-friendly quiz behavior
- Skin-goal product bundles
- Verified review band
- Subscription refill CTA
- Responsive drawer navigation

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Cormorant Garamond and DM Sans. Palette follows the brief exactly: cream, rose quartz, terracotta, cocoa, pearl, and sage.
