# Pinnacle Agency Portfolio

Premium standalone frontend skeleton for a creative agency portfolio.

## Includes
- Animated hero with CTA and live performance panel
- Service showcase with CSS icons
- Interactive case study filters
- Team cards with CSS avatars and social links
- Testimonials with star ratings
- Client logo wall built in CSS
- Contact form with project type and budget selectors
- Responsive hamburger menu with slide-in drawer

## Technical Notes
- Single-file HTML with inline CSS and JavaScript
- No runtime dependencies
- Google Fonts CDN only: Manrope and Inter
- Designed with the required color system
- Accessible landmarks, focus states, and reduced-motion handling
