# Pinnacle Agency Portfolio Design Book

## Concept
Pinnacle is a premium creative agency portfolio for teams that sell strategy, identity, digital experience, and launch campaigns. The first screen centers the agency proposition, while the rest of the page supports evaluation through services, case studies, team proof, testimonials, client marks, and a direct inquiry form.

## Typography
- Headings use Manrope from Google Fonts.
- UI text, body copy, controls, and labels use Inter from Google Fonts.
- Letter spacing remains neutral for a polished editorial feel.

## Color System
- Cloud `#F6F8FC` for the page background and elevated light panels.
- Ink `#19212B` for text, dark sections, and structural contrast.
- Indigo `#4F46E5` as the primary action and brand accent.
- Cyan `#06B6D4` for live indicators and secondary emphasis.
- Lime `#A3E635` for positive status and ratings.
- Coral `#FB7185` for warm campaign accents.

## Layout
- Sticky navigation with desktop links and a keyboard-navigable mobile drawer.
- Animated hero gradient with a performance panel.
- Four-card service grid with CSS-built icons.
- Interactive case study filters using accessible pressed states.
- Responsive team, testimonial, and client grids.
- Contact section combines qualification notes with a complete project inquiry form.

## Accessibility
- Semantic landmarks and section labels.
- Skip link for keyboard users.
- Focus-visible states across controls.
- Mobile drawer supports Escape, backdrop close, close button, and link close.
- Motion is reduced when the user prefers reduced motion.

## Content Voice
The language is direct, premium, and outcome-led. It avoids filler copy and gives each section a usable agency website narrative.
