# Hive Open Source Hub

Standalone open source project showcase template with repository listings, contribution graph, issue tracking, community cards, and responsive navigation.

## Files

- `index.html` - single-file HTML template with inline CSS and JavaScript
- `DESIGN_BOOK.md` - design rationale, colors, typography, and layout notes
- `README.md` - usage summary

## Features

- Sticky top navigation with search and New Repo action
- Mobile hamburger drawer
- KPI strip for repos, contributors, issues, and stars
- Repository grid with language badges, counts, last commit metadata, and contributor avatars
- CSS contribution heat map
- Issue tracker table with status and labels
- Community contributor cards

## Usage

Open `index.html` in a browser. The template has no build step and uses Google Fonts CDN only.
