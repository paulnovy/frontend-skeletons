# Echo Newsletter Platform
Standalone HTML skeleton for a newsletter publishing platform with subscriber management, email builder, analytics, and campaign scheduling.
## Files
- `index.html` includes inline CSS and JavaScript with Google Fonts only.
- `DESIGN_BOOK.md` documents the visual language and component approach.
- `README.md` describes the template contents.
## Preview
Open `index.html` in a browser. No build step is required.
## Included Sections
- Top navigation with brand, search, CTA, and mobile drawer.
- KPI strip for subscribers, open rate, click rate, and campaigns sent.
- Campaign list table.
- Subscriber management cards.
- Email builder preview.
- Campaign scheduler calendar.
- Analytics charts.
