# Haven Interior Studio - Design Book

## Project Name
Haven Interior Studio

## Product Type
Interior design studio and service business website.

## Intended Audience
Interior designers, home renovation studios, boutique service businesses, property stylists, and agencies building editorial lead-generation sites.

## User Problem
Homeowners often delay hiring a designer because the process feels expensive, vague, and risky. Haven makes the first step clear, guided, and low-pressure.

## UX Concept
The page feels like an interiors magazine but behaves like a service funnel: full-bleed hero, service tiers, three-step process, project stories, material palette, fit check, and booking CTA.

## Layout Structure
Sticky navigation, cinematic room hero, service cards, process timeline, case studies, material palette section, fit check cards, booking CTA, and footer.

## Typography System
Instrument Serif provides editorial room and project headlines. Inter acts as the Neue-style sans fallback for body copy, navigation, buttons, and service UI.

## Color System
Magnolia `#F4EBDD`, walnut `#5B3A29`, olive `#727B52`, plaster `#D8D1C4`, blackened bronze `#2B2520`, and clay `#B86B4B`.

## Component System
Core components include drawer navigation, service cards, process cards, case study cards, material swatches, fit check cards, and booking CTA.

## Interaction Rules
The booking path stays visible in nav and hero. Mobile drawer state updates `aria-expanded` and `aria-hidden`. The process section explains what happens after booking to reduce anxiety.

## Responsive Behavior
Desktop uses magazine-style composition. Tablet stacks editorial modules. Mobile uses stacked case-study cards and a simplified three-step process.

## Accessibility Notes
Semantic landmarks, skip link, visible focus states, descriptive links/buttons, accessible drawer state, readable contrast, and reduced-motion fallback are included.

## Performance Notes
Standalone HTML with inline CSS, minimal JavaScript, CSS-built visual areas, stable layout dimensions, and no heavy dependencies.

## Commercial Use Cases
Interior design studio homepage, renovation consultant funnel, project portfolio, material direction page, service booking landing page, and boutique agency template.
