# Haven Interior Studio

Magazine-style interior design studio skeleton with full-bleed room storytelling, service cards, guided process, case studies, material palette, fit check, and booking CTA.

## Includes

- Editorial full-bleed hero
- Service cards
- Three-step process section
- Project case studies
- Material palette swatches
- Fit check cards
- Booking CTA
- Responsive drawer navigation

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Instrument Serif and Inter. Palette follows the brief: magnolia, walnut, olive, plaster, blackened bronze, and clay.
