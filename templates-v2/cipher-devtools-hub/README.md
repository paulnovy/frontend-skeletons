# Cipher DevTools Hub

Standalone premium frontend skeleton for a developer tools marketplace with API documentation, SDK downloads, usage analytics, and platform activity.

## Files

- `index.html` contains the complete standalone page with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system, layout, responsive behavior, and accessibility approach.
- `README.md` records usage and verification notes.

## Usage

Open `index.html` directly in a browser. The page has no build step and no runtime dependency beyond the Google Fonts CDN request for Manrope and Inter.

## Verification

The required HTML parser, line count, palette, typography, and forbidden-text checks are intended to run after creation. Responsive verification should be performed with Playwright at desktop `1440`, tablet `768`, and mobile `390`, including a mobile screenshot for visual inspection.
