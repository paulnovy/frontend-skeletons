# Pulse Telemedicine

A standalone premium frontend skeleton for a telemedicine platform with scheduling, patient intake, video consultation support, prescription management, and patient queue operations.

## Files
- `index.html` contains the complete single-file skeleton with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system, component direction, and accessibility choices.

## Running
Open `index.html` in a browser. No build step is required. The only external dependency is the Google Fonts CDN for Manrope and Inter.

## Included Sections
- Top navigation with brand, links, search, notifications, and mobile drawer.
- KPI strip for the active clinical shift.
- Consultation scheduler with appointment cards and visit type badges.
- Patient intake form with symptoms, vitals, and history checklist.
- Video consultation workspace with patient sidebar and notes.
- Prescription management table with active and expired states.
- Waiting patient queue with wait times and priority badges.
