Interactive examples

Language tabs keep requests and responses side by side for fast scanning during implementation.

Request POST /runs
curl -X POST https://api.sage.dev/v3/projects/prj_42/runs \
  -H "Authorization: Bearer $SAGE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "input": { "customer_id": "cus_882" },
    "mode": "live"
  }'
Response 201 Created
{
  "id": "run_9Kb4",
  "status": "queued",
  "created_at": "2026-05-27T16:30:00Z",
  "links": { "self": "/runs/run_9Kb4" }
}
Request Python SDK
from sage import Client
client = Client(token=SAGE_TOKEN)
run = client.runs.create(
    project_id="prj_42",
    input={"customer_id": "cus_882"},
    mode="live",
)
Response Object
Run(
  id="run_9Kb4",
  status="queued",
  retries=0,
  project_id="prj_42"
)
Request JavaScript SDK
const sage = new SageClient({ token: process.env.SAGE_TOKEN });
const run = await sage.runs.create({
  projectId: "prj_42",
  input: { customer_id: "cus_882" },
  mode: "live"
});
Response Promise
{
  id: "run_9Kb4",
  status: "queued",
  retries: 0,
  projectId: "prj_42"
}

SDK guides

Install commands and tiny starts for the libraries maintained by the Sage platform team.

Python

Typed client with retries and async helpers.

Py
pip install sage-sdk
client = Client(token=SAGE_TOKEN)
project = client.projects.retrieve("prj_42")

JavaScript

ESM package for browsers, Node, and edge runtimes.

JS
npm install @sage/sdk
const sage = new SageClient({ token });
await sage.projects.retrieve("prj_42");

Go

Small transport layer with context-aware calls.

Go
go get github.com/sage/sdk-go
client := sage.New(token)
project, err := client.Project(ctx, "prj_42")

Ruby

Concise resource objects for Rails services.

Rb
bundle add sage-sdk
client = Sage::Client.new(token: token)
client.projects.retrieve("prj_42")
