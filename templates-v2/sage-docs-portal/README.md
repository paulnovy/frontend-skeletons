# Sage Docs Portal

A standalone premium frontend skeleton for an API documentation portal with interactive examples, SDK guides, endpoint reference cards, changelog history, and mobile documentation drawer.

## Files
- `index.html` contains the complete page, inline CSS, and inline JavaScript.
- `DESIGN_BOOK.md` explains typography, layout, color, and interaction choices.
- `README.md` summarizes usage and structure.

## Usage
Open `index.html` in a browser. The page has no build step and uses only Google Fonts as an external dependency.

## Included Sections
- Top navigation with brand, documentation links, search trigger, and version selector
- Hierarchical sidebar and mobile slide-in drawer
- API reference endpoint cards with method badges and parameter tables
- Tabbed code examples for cURL, Python, and JavaScript
- SDK guide cards with install commands and quick start snippets
- Changelog release timeline
- Full-screen search overlay grouped by content area
- Footer with visible studio credit

## Accessibility
The skeleton includes semantic landmarks, skip navigation, keyboard-operable drawer and search overlay, focus-visible styling, ARIA states for tabs and collapsible sections, and reduced-motion handling.
