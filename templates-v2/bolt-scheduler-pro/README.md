# Bolt Scheduler Pro

Premium frontend skeleton for an advanced scheduling tool with multi-calendar sync, meeting polls, availability management, booking, and integration monitoring.

## Files
- `index.html` contains the standalone HTML, inline CSS, and inline JavaScript.
- `DESIGN_BOOK.md` documents the visual system and interaction principles.

## Requirements Covered
- Top navigation with brand, links, search, and New Meeting action.
- Weekly calendar grid with CSS event blocks.
- Upcoming meeting list with avatars and status badges.
- Meeting poll cards with vote counts and deadlines.
- Availability manager with timezone selection and toggleable slots.
- Booking form with meeting type, duration, available slots, and attendee inputs.
- Google, Outlook, and Zoom integration cards.
- Mobile hamburger drawer with keyboard support.
