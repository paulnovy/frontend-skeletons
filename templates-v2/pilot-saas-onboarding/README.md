# Pilot SaaS Onboarding

A standalone premium frontend skeleton for a multi-step SaaS onboarding flow.

## Includes

- `index.html` with inline CSS and JavaScript.
- Google Fonts CDN for Manrope and Inter.
- Five-step onboarding wizard.
- Progress stepper and percentage badge.
- Profile form with upload area and toggles.
- Team invite list with add and remove actions.
- Integration cards with connect and disconnect states.
- Launch celebration with CSS confetti.
- Responsive hamburger menu with slide-in drawer.

## Usage

Open `index.html` in a browser. No build step or package install is required.
