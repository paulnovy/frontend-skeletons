# Shield Cybersecurity

A premium standalone frontend skeleton for a cybersecurity operations dashboard with threat monitoring, vulnerability scanning, incident response, score reporting, and asset inventory.

## Includes
- Single-file `index.html` with inline CSS and JavaScript.
- Google Fonts CDN loading for Manrope and Inter.
- Mobile hamburger drawer with keyboard support.
- Semantic sections for Dashboard, Threats, Vulnerabilities, Incidents, and Reports.
- Visible footer credit: `Shield Cybersecurity — designed by Meridian Risk Studio`.

## Run
Open `index.html` in a browser. No build step or package install is required.
