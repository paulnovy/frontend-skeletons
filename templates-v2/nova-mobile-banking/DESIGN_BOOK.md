# Nova Mobile Banking Design Book

Nova Mobile Banking is a mobile-first fintech skeleton for high-frequency banking actions, account monitoring, and card control.

## Visual System

- Typography: Manrope for headings and brand moments, Inter for interface copy and numbers.
- Colors: Cloud `#F6F8FC`, ink `#19212B`, indigo `#4F46E5`, cyan `#06B6D4`, lime `#A3E635`, coral `#FB7185`.
- Shape: compact rounded panels, touch-sized controls, and dense banking surfaces.
- Motion: lightweight drawer and offer rotation with reduced-motion support.

## Interaction Notes

- The hamburger opens a keyboard-navigable drawer with focus return.
- Balance visibility can be toggled without changing layout.
- Card freeze state updates visible status copy.
- Promotional offers rotate automatically and can be dismissed.

## Accessibility

- Semantic header, main, section, aside, nav, and footer landmarks are included.
- Interactive controls include labels or visible text.
- Focus-visible styling is defined globally.
- Reduced-motion preferences disable meaningful animation timing.
