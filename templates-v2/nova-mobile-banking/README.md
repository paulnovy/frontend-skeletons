# Nova Mobile Banking

Premium mobile-first banking frontend skeleton for touch-driven account overview, quick actions, transactions, card management, insights, and promotional offers.

## Files

- `index.html` - standalone HTML with inline CSS and JavaScript.
- `DESIGN_BOOK.md` - design system notes and behavior summary.
- `README.md` - usage overview.

## Requirements Covered

- Google Fonts CDN loads Manrope and Inter.
- Single-file runtime with no JavaScript or CSS dependencies.
- Mobile hamburger drawer, account balance hero, quick actions, transactions, card carousel, insights, rotating offers, and footer credit.
- Accessibility includes semantic landmarks, focus-visible states, keyboard drawer handling, and reduced-motion support.
