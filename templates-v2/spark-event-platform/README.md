# Spark Event Platform

A standalone premium HTML frontend skeleton for event management with ticket sales, speaker lineup, schedule building, and venue mapping.

## Included

- `index.html` with inline CSS and JavaScript
- `DESIGN_BOOK.md` with visual and interaction guidance
- Responsive top navigation and mobile drawer
- Filterable event listing grid
- Speaker cards with CSS gradient avatars
- ARIA-backed schedule tabs
- Ticket pricing tiers
- CSS venue floor plan

## Usage

Open `index.html` in a browser. The template has no build step and no runtime dependencies beyond the Google Fonts CDN.
