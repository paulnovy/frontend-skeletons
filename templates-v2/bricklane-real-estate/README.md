# Bricklane Real Estate

Real estate marketplace skeleton for comparing homes quickly with map/list search, saved search, property cards, mortgage estimates, neighborhood insights, and agent CTA.

## Includes

- Map/list search hero
- Property comparison cards
- Price, commute, monthly estimate, and deal tags
- Mortgage estimate panel
- Agent CTA
- Neighborhood editorial section
- Sticky mobile filter control

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Inter and Libre Baskerville. Palette follows the brief: stone, ink, brick, moss, sky, and cream.
