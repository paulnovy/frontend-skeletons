# Quant Analytics Lab

Standalone premium frontend skeleton for an advanced analytics workspace with data visualization, a SQL-like query builder, dashboard library, report generator, and connected data source monitoring.

## Files

- `index.html` contains the complete standalone page with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system, layout, responsive behavior, and accessibility approach.
- `README.md` records usage and verification notes.

## Usage

Open `index.html` directly in a browser. The page has no build step and no runtime dependency beyond the Google Fonts CDN request for Manrope and Inter.

## Verification

Run the required HTML parser, line count, palette, typography, and forbidden-text checks after creation. Responsive verification should be performed with Playwright at desktop `1440`, tablet `768`, and mobile `390`, including a mobile screenshot for visual inspection.
