# Grid Admin Panel Design Book

## Concept
Grid Admin Panel is a premium system administration skeleton for user management, role governance, configuration, audit review, and live system health.

## Visual System
- Headings use Manrope for confident operational hierarchy.
- Interface text uses Inter for dense, readable controls.
- The palette is limited to Cloud `#F6F8FC`, Ink `#19212B`, Indigo `#4F46E5`, Cyan `#06B6D4`, Lime `#A3E635`, and Coral `#FB7185` with opacity-based tints.
- Radius stays at 8px or below for a disciplined admin-tool feel.

## Layout
The page uses a sticky top navigation, KPI strip, primary work grid, user table, role cards, configuration form, audit log, and system health rail. Mobile navigation moves into a keyboard-accessible slide-in drawer.

## Accessibility
Semantic landmarks, labelled controls, focus-visible styling, scrollable audit region, keyboard drawer controls, and reduced-motion handling are included in the standalone HTML.
