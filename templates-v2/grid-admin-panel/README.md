# Grid Admin Panel

Premium standalone admin dashboard skeleton for managing users, roles, permissions, system configuration, audit activity, and infrastructure health.

## Files
- `index.html` contains all markup, CSS, and JavaScript.
- `DESIGN_BOOK.md` documents the visual system and layout decisions.

## Usage
Open `index.html` in a browser. No build step is required. The only external dependency is Google Fonts for Manrope and Inter.

## Features
- Sticky top navigation with search and admin avatar
- Mobile hamburger drawer with keyboard support
- KPI strip for users, sessions, approvals, and uptime
- Searchable user management table
- Role permission cards with toggles
- System configuration form
- Scrollable audit log
- System health meters
- Visible footer credit
