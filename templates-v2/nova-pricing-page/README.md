# Nova Pricing Page

A standalone premium frontend skeleton for a SaaS tiered pricing page.

## Included

- `index.html`: single-file HTML, CSS, and JavaScript implementation.
- `DESIGN_BOOK.md`: design rationale and interaction notes.
- Mobile hamburger drawer, billing cadence toggle, plan cards, comparison table, add-ons, enterprise callout, FAQ accordion, and footer credit.

## Requirements Met

- Manrope headings and Inter UI font from Google Fonts.
- Required color system with cloud, ink, indigo, cyan, lime, and coral.
- No runtime dependencies beyond Google Fonts.
- Accessible landmarks, keyboard controls, focus-visible styles, and reduced-motion support.
- Footer credit included.

Open `index.html` in a browser to view the skeleton.
