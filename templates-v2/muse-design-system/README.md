# Muse Design System

Premium frontend skeleton for design system documentation.

## Includes

- Token reference for color, typography, spacing, and layout.
- Component library cards for Button, Input, Card, Badge, Alert, and Modal.
- Icon set overview with usage context.
- Do and don't usage guidelines.
- Changelog and footer credit.
- Mobile hamburger drawer with accessible keyboard behavior.

## Usage

Open `index.html` in a browser or serve the folder with any static server.

## Constraints

- Single-file HTML with inline CSS and JavaScript.
- Google Fonts CDN is used for Manrope and Inter.
- No framework or build step is required.
