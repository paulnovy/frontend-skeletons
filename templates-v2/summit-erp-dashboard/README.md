# Summit ERP Dashboard

A standalone premium ERP dashboard skeleton for finance, inventory, HR, and procurement operations.

## Files
- `index.html` contains the full responsive dashboard with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system and interaction approach.
- `README.md` summarizes usage and structure.

## Features
- Sticky top navigation with brand, module switcher, search, notifications, and user profile.
- Mobile hamburger drawer with keyboard support.
- KPI strip for revenue, expenses, inventory value, headcount, procurement spend, and cash flow.
- Finance module with CSS bar chart and transactions table.
- Inventory module with low stock alerts, category levels, and warehouse summary.
- HR module with department headcount, reviews, leave requests, and turnover rate.
- Procurement module with purchase orders, supplier cards, and approval queue.
- Activity timeline with recent ERP events.

## Usage
Open `index.html` in a browser. The file is standalone apart from Google Fonts CDN requests for Manrope and Inter.
