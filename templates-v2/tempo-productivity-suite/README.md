# Tempo Productivity Suite

Standalone frontend skeleton for a focused productivity and work management suite.

## Files

- `index.html` - standalone HTML with inline CSS and JavaScript
- `DESIGN_BOOK.md` - design tokens, layout rules, and UX guidance
- `README.md` - usage notes

## Features

- Daily command center with focus time, blockers, and plan health
- State-first task board with filters and task detail drawer
- Calendar rail for today’s agenda
- Focus timer with protected-work actions
- Quick capture form with validation feedback
- Project progress, automation status, and workload health
- Keyboard movement across task cards
- Responsive hamburger navigation, visible focus states, and reduced-motion support

## Usage

Open `index.html` directly in a browser. The page uses Google Fonts for Space Grotesk and Inter; all styling and behavior is inline.

## Customization

- Edit CSS custom properties in `:root` to adjust palette, spacing, type, and motion.
- Duplicate `.task-card` elements to add work items.
- Update the `taskDetails` object in the inline script to map drawer content to cards.
- Tune project progress, agenda rows, and workload cells for your own operating model.
