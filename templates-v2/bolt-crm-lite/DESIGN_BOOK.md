# Bolt CRM Lite Design Book
## Product
Bolt CRM Lite is a compact sales workspace for contact management, deal tracking, activity review, and quick logging. The page is designed as a real working dashboard rather than a marketing page.
## Typography
- Headings use Manrope for sturdy, modern hierarchy.
- Interface text uses Inter for dense tables, cards, forms, and navigation.
- Letter spacing stays at `0` for a clean operational feel.
## Color System
- Cloud: `#F6F8FC` for the page background and quiet controls.
- Ink: `#19212B` for primary text and dark report cards.
- Indigo: `#4F46E5` for primary actions and brand emphasis.
- Cyan: `#06B6D4` for secondary highlights and email activity.
- Lime: `#A3E635` for positive pipeline states.
- Coral: `#FB7185` for risk and closed-lost signals.
## Layout
- Sticky top navigation keeps search and add-contact actions close at hand.
- KPI cards appear immediately below the workspace header for fast daily scanning.
- Contacts use a horizontal table with clear status badges.
- Deals use five Kanban columns with draggable cards.
- Activities and quick actions share the lower dashboard region.
- Mobile swaps the main navigation for a hamburger drawer.
## Interaction Notes
- Search filters contacts and deal cards from the top nav.
- Deal cards are draggable between pipeline stages.
- Quick action rows expand into compact forms for calls, emails, meetings, and notes.
- Toast feedback confirms lightweight UI actions.
## Accessibility
- Primary regions use clear labels and headings.
- Buttons include accessible names.
- The toast uses polite live-region behavior.
- Forms use visible labels.
