# Bolt CRM Lite
Lightweight CRM skeleton with contact management, deal pipeline, activity timeline, quick actions, and responsive mobile navigation.
## Files
- `index.html` — standalone page with inline CSS and JavaScript.
- `DESIGN_BOOK.md` — design rationale, layout notes, and interaction details.
- `README.md` — implementation summary.
## Features
- Sticky top nav with brand, section links, search, and add-contact CTA.
- KPI strip for contacts, active deals, pipeline value, and win rate.
- Contact list with company, email, phone, last activity, and status badges.
- Drag-and-drop Kanban pipeline with Lead, Qualified, Proposal, Closed Won, and Closed Lost stages.
- Activity timeline with call, email, meeting, and note events.
- Quick action accordions for calls, emails, meetings, and notes.
- Hamburger drawer for mobile screens.
## Usage
Open `index.html` in a browser. The template is self-contained except for Google Fonts.
