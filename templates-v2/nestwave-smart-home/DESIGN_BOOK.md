# Nestwave Smart Home - Design Book

## Project Name
Nestwave Smart Home

## Product Type
Smart home control dashboard.

## Intended Audience
Smart home product teams, IoT dashboard designers, device manufacturers, home automation startups, and frontend teams building control surfaces.

## User Problem
Smart home users need device states to be obvious. They want room-level control, reliable security visibility, energy feedback, and quick scene activation without hunting through nested menus.

## UX Concept
Nestwave is an app shell focused on clarity: home overview, room tabs, energy card, room cards with large toggles, security panel, automation timeline, and one-hand-friendly scene buttons.

## Layout Structure
Desktop uses a sidebar app shell with main dashboard content. Mobile hides the sidebar, uses room tabs, large toggles, and a sticky scene shortcut.

## Typography System
Manrope handles UI labels, body copy, controls, and status text. Space Grotesk gives device headings and dashboard moments a clear smart-home identity.

## Color System
Night `#0F172A`, warm light `#FFE8A3`, sky `#93C5FD`, mint `#A7F3D0`, clay `#D97706`, and soft gray `#E2E8F0`.

## Component System
Core components include sidebar navigation, home status card, room tabs, energy card, room cards, toggles, security state panel, automation timeline, scene buttons, and sticky mobile scenes.

## Interaction Rules
Device state is shown with text and toggle position. Room tabs and scenes update active state. Controls are large enough for touch-first use.

## Responsive Behavior
Desktop uses room overview and dashboard panels. Mobile uses large controls, stacked rooms, and bottom scene access for one-handed activation.

## Accessibility Notes
Semantic landmarks, skip link, visible focus states, descriptive toggle labels, readable contrast, clear text states, and reduced-motion fallback are included.

## Performance Notes
Standalone HTML with inline CSS, minimal JavaScript, no dependencies, stable dashboard dimensions, and CSS-only controls.

## Commercial Use Cases
Smart home app prototype, IoT dashboard, home automation control center, device management UI, security panel, and energy monitoring dashboard.
