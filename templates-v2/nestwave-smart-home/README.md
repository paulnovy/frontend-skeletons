# Nestwave Smart Home

Smart home control dashboard skeleton with room grouping, energy card, device status, automation timeline, security panel, and one-hand-friendly scene presets.

## Includes

- Home overview app shell
- Room tabs and room cards
- Energy summary
- Large device toggles
- Security panel
- Automation timeline
- Scene presets
- Sticky mobile scene shortcut

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Manrope and Space Grotesk. Palette follows the brief: night, warm light, sky, mint, clay, and soft gray.
