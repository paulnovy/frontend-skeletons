# Atlas Travel Planner

Premium standalone frontend skeleton for a travel planning tool with itinerary building, destination discovery, booking management, saved destinations, and budget tracking.

## Files

- `index.html` — single-file HTML, CSS, and JavaScript implementation.
- `DESIGN_BOOK.md` — design system, layout notes, and accessibility summary.
- `README.md` — implementation overview.

## Requirements Covered

- Manrope headings and Inter UI loaded from Google Fonts.
- Cloud, ink, indigo, cyan, lime, and coral color system.
- Responsive top navigation with keyboard-navigable mobile drawer.
- Hero search, featured destinations, itinerary timeline, discovery grid, booking list, saved destinations row, CSS-only budget donut, and footer credit.
- No runtime dependencies beyond Google Fonts.

## Preview

Open `index.html` directly in a browser or serve the repository with the existing Vite setup.
