# Craft Careers Page Design Book

## Concept

Craft Careers Page is a premium HR recruitment skeleton for a company careers page. It combines a confident hiring story, culture values, practical benefits, filterable job listings, and a three-step application flow in one polished page.

## Visual System

- Typography: Manrope for headings and Inter for interface text, loaded through Google Fonts.
- Page background: Cloud `#F6F8FC`.
- Primary text and dark panels: Ink `#19212B`.
- Primary accent: Indigo `#4F46E5`.
- Secondary accent and live indicators: Cyan `#06B6D4`.
- Healthy state: Lime `#A3E635`.
- Alert state: Coral `#FB7185`.

The page uses only the required palette plus opacity-based tints. Cards keep an 8px radius, restrained borders, and soft shadows for a premium operational feel.

## Sections

- Top navigation with desktop links, primary CTA, and accessible mobile drawer.
- Hero with “Join Our Team,” supporting hiring copy, metrics, and a CSS gradient team photo area.
- Culture showcase with value cards, icons, and short descriptions.
- Benefits grid covering health, remote work, learning, equity, recharge, tools, mobility, and family support.
- Filterable job listings with department badges, location, job type, and apply actions.
- Multi-step application form for resume upload, cover letter, and review.
- Employee testimonials with visual photo areas and real quote copy.
- Footer with visible studio credit and social links.

## Interaction Notes

- Mobile hamburger opens a slide-in drawer with keyboard escape support.
- Job filters update visible roles and announce the count.
- Application progress updates as the user moves through the form.
- Focus-visible styles are defined globally.
- Reduced-motion preferences are honored.

## Accessibility

Semantic landmarks include header, nav, main, sections, aside, form, and footer. The page includes a skip link, aria labels for navigation and job lists, keyboard-friendly buttons, visible focus states, and readable contrast across light and dark surfaces.
