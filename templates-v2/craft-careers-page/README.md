# Craft Careers Page

A standalone premium frontend skeleton for a company careers page in the HR and recruitment category.

## Includes

- Sticky top navigation with mobile hamburger drawer
- Hero section with gradient team photo area
- Culture showcase cards
- Benefits cards
- Filterable job listings
- Three-step application flow
- Employee testimonials
- Footer with visible design credit

## Tech

This template is a single `index.html` file with inline CSS and JavaScript. The only external dependency is Google Fonts for Manrope and Inter.

## Usage

Open `index.html` in a browser. No build step is required.
