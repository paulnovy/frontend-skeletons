# Flux Marketplace Design Book

## Concept
Flux Marketplace is a premium digital goods marketplace for creators, teams, and independent sellers. The interface prioritizes fast discovery, clear seller trust signals, and lightweight commerce controls.

## Visual System
- Typography: Manrope for high-impact headings, Inter for navigation, controls, body copy, and data labels.
- Palette: Cloud `#F6F8FC`, ink `#19212B`, indigo `#4F46E5`, cyan `#06B6D4`, lime `#A3E635`, coral `#FB7185`.
- Shape: compact 8px radius panels and cards with restrained borders and soft shadows.
- Imagery: CSS gradient thumbnails create premium digital product art without external assets.

## Layout
The skeleton uses a sticky top navigation, featured collection hero, metric strip, two-column marketplace browse area, seller profile band, review system, seller callout, cart drawer, and mobile navigation drawer.

## Accessibility
Semantic landmarks, labeled controls, focus-visible treatment, Escape drawer close behavior, keyboard reachable drawer buttons, and reduced-motion support are included.

## Components
- Product cards with category tags, seller avatars, prices, ratings, and add buttons.
- Category sidebar with counts and expandable niche list.
- Seller cards with sales metrics, ratings, and specialty tags.
- Review cards with reviewer, date, rating, and concise review text.
- Cart preview drawer with quantities, subtotal, and checkout action.
