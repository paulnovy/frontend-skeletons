# Pulse Mental Health Design Book

Pulse Mental Health is a calm care workspace for people tracking mood, therapy, journaling, and wellness resources. The interface favors clear hierarchy, grounded color, and repeated utility patterns over decorative marketing composition.

## Visual System

- Headings use Manrope for confident, rounded structure.
- UI text uses Inter for readable scanning in cards, forms, and navigation.
- Cloud `#F6F8FC` creates the page surface.
- Ink `#19212B` anchors text and the breathing panel.
- Indigo `#4F46E5` identifies primary action and therapy status.
- Cyan `#06B6D4` supports calm resource and progress moments.
- Lime `#A3E635` marks positive streaks and brighter mood days.
- Coral `#FB7185` highlights low mood days and saved resource warmth.

## Layout

The page is a standalone dashboard with sticky navigation, mobile hamburger drawer, and full-width content sections. Repeated cards use 8px radii, restrained borders, and stable grid sizing so dense mental-health information remains easy to scan.

## Components

- Mood tracker: five-point emoji selector, note field, save state, and 30-day calendar.
- Therapy sessions: upcoming appointment list with therapist initials, session type badges, and action buttons.
- Journal: reflection cards with dates, mood badges, previews, and tags.
- Wellness resources: resource cards with type badges, duration, category, and bookmark toggle.
- Progress dashboard: weekly mood chart plus streak, journal, and sessions attended metrics.

## Responsive Behavior

Desktop uses two-column care panels and four-column resources. Tablet collapses the navigation into a drawer and resources into two columns. Mobile presents one column, compact mood controls, accessible appointment actions, and a full-height drawer.
