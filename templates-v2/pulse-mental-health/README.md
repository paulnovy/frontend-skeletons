# Pulse Mental Health

A standalone single-page mental health platform skeleton with inline CSS and JavaScript.

## Includes

- Sticky top navigation with brand, section links, search, streak badge, and mobile drawer.
- Mood tracker with five mood states, notes field, save feedback, and 30-day color-coded calendar.
- Therapy sessions with upcoming appointments, therapist names, dates, type badges, and history.
- Journal cards with dates, mood badges, previews, tags, and a write-new action.
- Wellness resources with article, video, and exercise cards plus bookmark toggles.
- Progress dashboard with weekly mood chart and care metrics.
- Footer credit: `Pulse Mental Health — designed by Studio`.

## Files

- `index.html` contains the full standalone experience.
- `DESIGN_BOOK.md` documents the visual system and layout choices.
- `README.md` summarizes usage and structure.
