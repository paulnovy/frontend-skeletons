# Nova Fintech Banking

Premium standalone frontend skeleton for a neobanking interface with account overview, transaction history, transfer approval, spending analytics, and virtual card management.

## Files

- `index.html` - single-file HTML with inline CSS and JavaScript.
- `DESIGN_BOOK.md` - design rationale, palette usage, component notes, and accessibility decisions.
- `README.md` - implementation summary.

## Requirements Covered

- Manrope headings and Inter UI text loaded from Google Fonts CDN.
- Cloud, ink, indigo, cyan, lime, and coral palette with tints through opacity.
- Top nav, account overview, linked accounts, quick actions, scrollable transactions, CSS-only donut chart, transfer flow, card management, and visible footer credit.
- Mobile hamburger drawer with keyboard support, Escape close, aria state updates, focus-visible styling, and reduced motion handling.
- No external dependencies beyond Google Fonts.

## Preview

Open `index.html` directly in a browser or serve the `frontend-skeletons` workspace with Vite.
