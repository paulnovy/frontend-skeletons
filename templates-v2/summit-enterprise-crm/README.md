# Summit Enterprise CRM

Premium standalone frontend skeleton for an enterprise CRM with pipeline visualization, contact management, deal tracking, activity history, and report previews.

## Files

- `index.html`: single-file HTML, inline CSS, and inline JavaScript.
- `DESIGN_BOOK.md`: design rationale, visual system, layout, and interaction notes.
- `README.md`: implementation overview.

## Features

- Sticky top navigation with brand, links, search, and notification badge.
- Mobile hamburger drawer with keyboard navigation support.
- KPI strip for total deals, pipeline value, win rate, and average deal cycle.
- Drag-and-drop pipeline Kanban with Lead, Qualified, Proposal, Negotiation, Won, and Lost stages.
- Searchable contact table with enterprise account details.
- Expandable deal detail cards with value, stage, owner, next action, and close date.
- Activity feed for calls, emails, meetings, and notes.
- CSS-only charts for deal velocity and conversion funnel.
- Visible footer credit for Northstar Systems Studio.

## Design System

Typography loads Manrope and Inter from Google Fonts. The palette uses cloud, ink, indigo, cyan, lime, and coral with soft opacity treatments for enterprise-grade contrast and status signaling.

## Usage

Open `index.html` in a browser. No build step or package installation is required.
