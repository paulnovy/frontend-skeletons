# Hive Forum Classic

Premium standalone frontend skeleton for a classic online forum.

## Included

- `index.html` with inline CSS and JavaScript
- Google Fonts loading for Manrope and Inter
- Responsive top navigation with mobile hamburger drawer
- Category listing, sortable thread table, nested thread view, reputation sidebar, moderation tools, and new thread form
- Visible footer credit

## Usage

Open `index.html` in a browser. No build step is required.

## Verification

The template is designed to pass the requested HTML parser, line count, font, color, and content checks.
