# Chargegrid EV Network

Map-first EV charging network skeleton focused on charger availability, speed, cost, reliability, route planning, pricing, and fleet dashboard visibility.

## Includes

- Map-first hero
- Live station cards
- Charger status chips
- Route planner
- Pricing card
- Fleet dashboard preview
- Mobile charger bottom sheet
- Responsive drawer navigation

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Rajdhani and Inter. Palette follows the brief: asphalt, electric green, charge blue, white, gray, and amber.
