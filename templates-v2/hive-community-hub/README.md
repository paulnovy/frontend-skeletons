# Hive Community Hub

Premium standalone frontend skeleton for an online community platform with forums, member directories, trending topics, community guidelines, and event management.

## Files

- `index.html` — single-file HTML, inline CSS, and inline JavaScript implementation.
- `DESIGN_BOOK.md` — design system, layout notes, content model, and accessibility summary.
- `README.md` — implementation overview and usage notes.

## Requirements Covered

- Manrope headings and Inter UI text loaded from Google Fonts CDN.
- Cloud, ink, indigo, cyan, lime, and coral color system with opacity-based tints.
- Top navigation, KPI strip, forum threads, searchable member directory, event cards, trending topic cloud, expandable guidelines, and footer credit.
- Keyboard-navigable mobile hamburger drawer with Escape close behavior and visible focus states.
- No runtime dependencies beyond Google Fonts.

## Preview

Open `index.html` directly in a browser or serve the repository with the existing Vite setup.
