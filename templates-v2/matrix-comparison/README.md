# Matrix Comparison

A standalone premium frontend skeleton for SaaS product comparison workflows.

## Included
- `index.html` with inline CSS and JavaScript
- Google Fonts for Manrope and Inter
- Responsive navigation with keyboard-friendly mobile drawer
- Product selector, feature matrix, pricing comparison, recommendation engine, user review summaries, and footer credit

## Use
Open `index.html` in a browser. No build step is required.

## Notes
The file is dependency-free except for the Google Fonts CDN request. The design uses the required Cloud, Ink, Indigo, Cyan, Lime, and Coral color system with opacity-based tints.
