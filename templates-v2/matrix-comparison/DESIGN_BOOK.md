# Matrix Comparison Design Book

## Intent
Matrix Comparison is a premium SaaS comparison workspace for buyers who need to evaluate products across features, pricing, recommendations, and reviews.

## Typography
- Headings use Manrope with strong weight and compact line height.
- Interface text uses Inter for dense controls, matrices, cards, and navigation.

## Color System
- Cloud `#F6F8FC` is the page background and light surface anchor.
- Ink `#19212B` is used for text, dark panels, and high-contrast controls.
- Indigo `#4F46E5` is the primary action and recommendation accent.
- Cyan `#06B6D4` marks live status and interactive focus.
- Lime `#A3E635` communicates positive feature availability and healthy status.
- Coral `#FB7185` communicates removal, risk, and unavailable features.

## Layout
The skeleton uses a sticky top navigation, a decisive hero, three product slots, a scrollable feature matrix with sticky first column, pricing cards, recommendation cards, review cards, and a concise footer credit.

## Accessibility
The page includes semantic landmarks, skip navigation, visible focus states, accessible drawer controls, keyboard dismissal with Escape, and reduced motion support.
