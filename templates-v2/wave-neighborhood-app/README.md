# Wave Neighborhood App

Premium standalone frontend skeleton for a hyperlocal neighborhood social app.

## Included
- `index.html`: single-file HTML, CSS, and JavaScript implementation.
- `DESIGN_BOOK.md`: design rationale and system notes.

## Features
- Community feed with post cards and social actions.
- Neighbor directory with interests and connection states.
- Local business directory with ratings, hours, distance, and phone details.
- Event board with dates, locations, hosts, attendee counts, and RSVP actions.
- Safety alerts with advisory, warning, and critical severities.
- Neighborhood stats panel and keyboard-friendly mobile drawer.

Open `index.html` in a browser to view the skeleton.
