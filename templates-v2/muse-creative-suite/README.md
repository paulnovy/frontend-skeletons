# Muse Creative Suite

A standalone premium frontend skeleton for creative project management.

## Includes

- Top navigation with brand, search, section links, and New Project action.
- Kanban project board with draggable cards.
- Asset library with image, video, audio, and document cards.
- Team collaboration cards with live status indicators.
- Template gallery with usage counts and action buttons.
- Activity timeline for uploads, comments, approvals, and assignments.
- Project detail panel with progress, budget, deadline, assignees, and checklist.
- Mobile hamburger menu with slide-in drawer.

## Usage

Open `index.html` in a browser. The file is self-contained and uses Google Fonts as its only external dependency.

## Design

The skeleton uses Manrope for headings, Inter for UI text, and a focused creative palette of cloud, ink, indigo, cyan, lime, and coral.
