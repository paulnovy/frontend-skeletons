# Muse Creative Suite Design Book

Muse Creative Suite is a premium creative workspace skeleton for project boards, asset operations, and collaborative production review.

## Visual System

- Typography: Manrope is used for expressive headings and brand moments. Inter is used for dense interface text.
- Palette: Cloud `#F6F8FC`, ink `#19212B`, indigo `#4F46E5`, cyan `#06B6D4`, lime `#A3E635`, and coral `#FB7185`.
- Shape: Interface elements use restrained 8px radii for a polished operational product feel.
- Surfaces: Cloud backgrounds, white glass panels, ink hero panels, and low-opacity tints keep the workspace bright and organized.

## Experience Notes

- The first screen behaves like a real studio dashboard rather than a marketing page.
- Kanban cards are draggable across production columns.
- Mobile navigation uses an accessible slide-in drawer with focus handling and Escape support.
- Asset thumbnails use CSS gradients so the skeleton stays standalone.
- Status, timeline, and checklist patterns are ready to connect to product data.

## Accessibility

- Semantic header, main, section, article, aside, nav, and footer landmarks.
- Skip link for keyboard users.
- Visible focus states and reduced-motion support.
- Drawer controls expose `aria-expanded`, `aria-hidden`, and descriptive labels.
