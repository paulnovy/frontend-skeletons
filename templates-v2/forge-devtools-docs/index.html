Quickstart

Get up and running with Forge in under 30 seconds. This guide walks you through installation, project setup, and your first deployment.

Prerequisites

Make sure you have Node.js 18+ and npm 9+ installed. You'll also need a Forge account — sign up free.

Install the CLI

npm install -g @forge/cli
forge login
Note: If you're using macOS, you may need to add sudo for global installs. Or use npx instead.

Create a project

forge init my-app
cd my-app
forge dev

This starts a local development server at http://localhost:3000 with hot reload enabled.

Deploy to production

forge deploy --prod

Your app is now live. Forge handles SSL, CDN, and scaling automatically.
