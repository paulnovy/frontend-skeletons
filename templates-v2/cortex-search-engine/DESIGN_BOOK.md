# Cortex Search Engine Design Book

## Concept
Cortex Search Engine is a premium enterprise search interface for teams that need trusted discovery across policy, people, operations, and knowledge repositories. The experience emphasizes fast scanning, transparent ranking, and confidence signals.

## Visual System
- Headings use Manrope for a precise, executive tone.
- UI text uses Inter for dense operational readability.
- Cloud `#F6F8FC` establishes a quiet workspace background.
- Ink `#19212B` anchors navigation, preview surfaces, and readable copy.
- Indigo `#4F46E5` identifies primary search actions and active state.
- Cyan `#06B6D4` marks live intelligence and secondary discovery signals.
- Lime `#A3E635` communicates healthy ranking and approved status.
- Coral `#FB7185` flags alerts and critical analytics.

## Layout
The interface uses a three-zone enterprise workspace: filters on the left, result discovery in the center, and preview plus analytics on the right. On mobile, filters move into a keyboard-accessible slide-in drawer and the preview stacks beneath results.

## Interaction Notes
- Search suggestions expose autocomplete, related queries, and correction actions.
- Filter groups collapse independently.
- Result cards update the preview panel.
- Ranking controls show active ranking intent.
- The mobile drawer supports escape-to-close and backdrop closing.

## Accessibility
The page uses semantic landmarks, labelled controls, focus-visible states, accessible drawer toggles, a skip link, and reduced-motion handling.
