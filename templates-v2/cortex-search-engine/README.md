# Cortex Search Engine

Premium frontend skeleton for an enterprise search interface with filters, facets, intelligent ranking controls, result preview, and search analytics.

## Files
- `index.html` contains the standalone UI with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system and interaction model.

## Features
- Top navigation with brand, search, voice control, filters, sort, and result count.
- Collapsible filter sidebar for date range, source, type, author, and language.
- Faceted navigation chips with active state.
- Intelligent suggestions with related queries, correction flow, and autocomplete.
- Result cards with highlighted terms, source badges, dates, and relevance scores.
- Result preview panel with metadata, key phrases, and document outline.
- Search analytics cards and top queries table.
- Mobile hamburger drawer with keyboard support.

## Technical Notes
The skeleton is a single HTML file and only loads Manrope and Inter from Google Fonts CDN. It uses no JavaScript or CSS frameworks.
