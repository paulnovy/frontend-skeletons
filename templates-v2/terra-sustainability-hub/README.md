# Terra Sustainability Hub

A standalone premium frontend skeleton for a corporate sustainability and ESG reporting dashboard.

## Files
- `index.html` contains all HTML, CSS, and JavaScript.
- `DESIGN_BOOK.md` documents the visual and interaction system.

## Usage
Open `index.html` in a browser. The page has no build step and uses only Google Fonts as an external dependency.

## Features
- ESG KPI strip and scorecards
- Carbon footprint CSS bar chart
- Green initiative and sustainability goal cards
- Report download card layout
- Mobile hamburger drawer
- Accessibility-focused landmarks, focus states, and reduced-motion support
