# Pulse Health & Fitness

Standalone premium frontend skeleton for a consumer fitness tracking app with workout plans, nutrition logging, recovery goals, and progress visualization.

## Files

- `index.html` contains the complete standalone app screen with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the visual system, layout structure, responsive behavior, and accessibility approach.
- `README.md` records usage and verification notes.

## Usage

Open `index.html` directly in a browser. The skeleton has no build step and no runtime dependency beyond the Google Fonts CDN request for Manrope and Inter.

## Verification

Run the required HTML parser, line count, palette, font, and forbidden-text checks after creation. Responsive verification should be performed with Playwright at desktop `1440`, tablet `768`, and mobile `390`, including a mobile screenshot for visual inspection.
