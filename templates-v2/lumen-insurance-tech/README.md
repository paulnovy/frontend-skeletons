# Lumen Insurance Tech

Standalone HTML skeleton for an insurance management platform with policy management, claims processing, risk assessment, and customer portal views.

## Files

- `index.html` contains the full single-page interface with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the design concept, palette, typography, layout, and content model.

## Features

- Sticky top navigation with brand, links, search, and New Policy action.
- Mobile hamburger navigation with slide-out drawer.
- KPI strip for active policies, open claims, total premiums, and loss ratio.
- Policy table with type badges, status indicators, premiums, and expiry dates.
- Claims Kanban pipeline for Submitted, Under Review, Approved, Denied, and Paid states.
- Risk assessment cards with score bars, mitigation status, and review dates.
- Customer portal cards showing policy counts, claims history, and risk badges.

## Usage

Open `index.html` in a browser. The page is standalone and only loads Google Fonts from a CDN.
