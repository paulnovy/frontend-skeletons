# Shield Privacy Dashboard

Premium frontend skeleton for a privacy management dashboard covering consent tracking, data mapping, subject rights workflows, compliance reporting, and risk assessment.

## Files

- `index.html`: standalone dashboard with inline CSS and JavaScript.
- `DESIGN_BOOK.md`: visual system and implementation notes.
- `README.md`: project overview.

## Features

- Sticky top navigation with search and notification badge.
- Mobile hamburger drawer with keyboard support.
- KPI strip for consent rate, open requests, processing activities, and compliance score.
- Consent and subject request tables.
- Data flow, compliance report, and risk assessment cards.
- Google Fonts typography using Manrope and Inter.

## Usage

Open `index.html` in a browser. No build step or package installation is required.
