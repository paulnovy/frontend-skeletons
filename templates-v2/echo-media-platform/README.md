# Echo Media Platform

A standalone premium frontend skeleton for a digital media publication with article layouts, podcast integration, trending stories, category filters, and newsletter signup.

## Files
- `index.html` contains all HTML, CSS, and JavaScript in one file.
- `DESIGN_BOOK.md` documents the design direction and accessibility notes.

## Usage
Open `index.html` in a browser. The page loads Manrope and Inter from Google Fonts and has no other external dependencies.

## Included Sections
- Top navigation with brand, links, search, subscribe CTA, and mobile drawer
- Hero article with gradient media area and editorial metadata
- Article grid with gradient thumbnails and author/date metadata
- Podcast episode cards with play buttons and durations
- Newsletter signup forms with privacy notes
- Trending sidebar with ranked most-read articles
- Horizontal category tag strip
- Footer with credit, social links, and navigation
