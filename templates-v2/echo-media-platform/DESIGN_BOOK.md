# Echo Media Platform Design Book

## Concept
Echo Media Platform is a premium digital media publication skeleton for editorial teams that publish articles, podcasts, and newsletters from one polished front page.

## Visual System
- Headings use Manrope for a strong magazine voice.
- Interface text uses Inter for clear navigation, forms, metadata, and controls.
- The palette is restricted to Cloud `#F6F8FC`, Ink `#19212B`, Indigo `#4F46E5`, Cyan `#06B6D4`, Lime `#A3E635`, and Coral `#FB7185`, with opacity-based tints.

## Layout Notes
- Sticky top navigation supports desktop links, search, and a subscribe action.
- Mobile uses a keyboard-accessible slide-in drawer with Escape and backdrop close behavior.
- The hero article uses a CSS gradient image area so the skeleton is fully standalone.
- Article cards include category, headline, summary, author initials, and date.
- Podcast cards include episode number, duration, description, and play controls.
- Newsletter forms appear as both a sticky sidebar card and a wide signup module.

## Accessibility
- Semantic landmarks and labelled navigation are included.
- Focus-visible states are prominent.
- Drawer state uses `aria-expanded` and `aria-hidden`.
- Reduced motion preferences are respected.
