# Forge Dev Portal

Premium standalone frontend skeleton for a developer portal with API playground, webhook management, API key administration, usage analytics, and recent activity.

## Files

- `index.html` contains all markup, CSS, and JavaScript in one file.
- `DESIGN_BOOK.md` documents the visual system and section intent.

## Requirements Met

- Uses Manrope and Inter from Google Fonts.
- Uses the required Cloud, ink, indigo, cyan, lime, and coral color system.
- Includes mobile hamburger navigation with a slide-in drawer.
- Includes semantic landmarks, accessible labels, focus-visible styles, and reduced-motion support.
- Contains the required visible footer credit.

Open `index.html` in a browser to view the skeleton.
