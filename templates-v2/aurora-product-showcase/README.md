# Aurora Product Showcase

A standalone premium frontend skeleton for a single-product marketing page.

## Included
- Sticky top navigation with mobile hamburger drawer
- Hero section with product name, tagline, price, visual product stage, and CTAs
- Alternating feature deep-dives
- Specification grid
- CSS gradient gallery with hover zoom
- Verified customer review cards
- Purchase flow with variants, quantity, warranty options, and live total
- Footer credit

## Usage
Open `index.html` in a browser. The file is self-contained except for Google Fonts.

## Constraints Met
- Single HTML file with inline CSS and JavaScript
- Manrope and Inter loaded from Google Fonts CDN
- Required color system used throughout
- No external JavaScript or CSS dependencies
- Accessible mobile drawer, skip link, focus-visible styles, and reduced motion support
