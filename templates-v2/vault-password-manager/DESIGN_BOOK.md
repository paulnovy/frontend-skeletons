# Vault Password Manager — Design Book

## Concept
Password vault and credential management dashboard. Secure storage, password generator, breach monitoring, team sharing.

## Layout
- Sticky top nav: brand, Vault/Generator/Breach Monitor/Teams/Settings, search, CTA "Add Credential"
- KPI strip: stored credentials, weak passwords, breach alerts, team members
- Credential list table with category badges, strength bars
- Password generator with options
- Breach monitoring alerts
- Team sharing section
- Footer

## Design System
Colors: Cloud #F6F8FC, Ink #19212B, Indigo #4F46E5, Cyan #06B6D4, Lime #A3E635, Coral #FB7185
Fonts: Manrope 700/800 headings, Inter 400/500 body
Mobile: hamburger + drawer, responsive tables
