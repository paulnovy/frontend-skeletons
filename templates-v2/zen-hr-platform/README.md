# Zen HR Platform

Premium standalone frontend skeleton for HR management software.

## Included

- Employee directory with search and filters
- Leave management table with status badges and action buttons
- Performance review cycle cards with progress bars
- Simplified CSS org chart
- HR announcements panel
- Responsive top navigation and mobile drawer

## Technical Notes

- Single-file HTML implementation with inline CSS and JavaScript
- No external dependencies except Google Fonts
- Uses Manrope for headings and Inter for UI text
- Accessible focus states, semantic landmarks, and reduced-motion handling
