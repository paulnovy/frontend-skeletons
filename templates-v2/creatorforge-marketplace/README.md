# CreatorForge Marketplace

CreatorForge Marketplace is a standalone creator economy marketplace homepage skeleton.

It includes:
- bold creator discovery hero
- sticky category filter chips
- featured creator match board
- creator cards with price, availability, and audience fit
- service bundle cards
- social proof strip
- booking CTA panel
- mobile vertical creator cards

Fonts: Syne and DM Sans.

Palette: #080808, #FF3D9A, #F9F871, #3B82F6, #FFFFFF, #C4B5FD.
