# Verdant Climate Platform

Trust-heavy climate tech SaaS skeleton that turns emissions, supplier evidence, compliance deadlines, audit trail, and ROI into a clear business action flow.

## Includes

- Emissions dashboard preview
- Regulatory timeline
- Supplier evidence cards
- Audit trail section
- ROI calculator CTA
- Sticky mobile emissions summary
- Responsive drawer navigation

## Tech

Standalone HTML with inline CSS and minimal JavaScript. Fonts: Space Grotesk, Inter, and IBM Plex Mono. Palette follows the brief: deep green, mist, leaf, cyan, earth, and ink.
