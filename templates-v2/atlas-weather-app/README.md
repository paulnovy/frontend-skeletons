# Atlas Weather App

Standalone premium frontend skeleton for a weather utility app with forecasts, radar, severe weather alerts, and air quality.

## Files

- `index.html` contains all HTML, CSS, and JavaScript in one file.
- `DESIGN_BOOK.md` documents the visual system, interactions, content model, and accessibility decisions.

## Requirements Covered

- Manrope headings and Inter UI text from Google Fonts CDN.
- Required color system with cloud, ink, indigo, cyan, lime, and coral.
- Top navigation, current conditions, hourly forecast, 7-day forecast, radar map, alerts, air quality, and footer credit.
- Mobile hamburger menu with slide-in drawer, Escape close, focus return, and focus trapping.
- `prefers-reduced-motion` support.

## Usage

Open `index.html` in any modern browser. No build step or external runtime is required.
