# Bolt Logistics Dashboard

Premium standalone frontend skeleton for a logistics management dashboard.

## Included
- `index.html`: single-file implementation with inline CSS and JavaScript.
- `DESIGN_BOOK.md`: design rationale, typography, color system, layout notes, and accessibility notes.
- `README.md`: project overview.

## Features
- Supply chain dashboard with KPI strip.
- Shipment tracking table with status badges.
- Warehouse overview cards with capacity bars.
- CSS route map visualization.
- CSS-only delivery analytics chart.
- Exception alert list.
- Responsive mobile hamburger drawer.

## Constraints
- No runtime dependencies beyond Google Fonts.
- Uses Manrope for headings and Inter for interface text.
- Color palette follows the required cloud, ink, indigo, cyan, lime, and coral system.
