# Hearth Community Hub

Hearth Community Hub is a standalone community platform homepage skeleton.

It includes:
- welcome hero
- today-first event calendar
- discussion highlights
- member spotlights
- learning paths
- contribution prompts
- mobile priority chips for events and discussions

Fonts: Nunito Sans and Fraunces.

Palette: #FFF8EA, #1F4D3A, #FF6B6B, #FBBF24, #BAE6FD, #1F2937.
