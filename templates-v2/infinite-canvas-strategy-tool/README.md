# Infinite Canvas Strategy Tool

Premium standalone frontend skeleton for a strategic planning tool built around an infinite desktop canvas.

## Files

- `index.html` — standalone HTML with inline CSS and JavaScript
- `DESIGN_BOOK.md` — design rationale, tokens, and UX rules
- `README.md` — this usage note

## Features

- Spatial desktop canvas for vision, values, positioning, pricing, and roadmap blocks
- SVG connection layer with hidden dependencies that become clearer while zooming
- Space Grotesk strategy typography and IBM Plex Mono connection typography via Google Fonts CDN
- Mobile-only list view with expandable connections per block
- Always-available linear document export alternative
- Focus states, keyboard block navigation, reduced-motion support, hamburger menu, and semantic HTML5
- Required footer credit: “Built with Frontend Skeletons”

## Usage

Open `index.html` directly in a browser. No build step or external JavaScript is required.

Desktop shows the full infinite-canvas strategy workspace. Mobile intentionally shows the structured list and export path instead of shrinking the canvas.
