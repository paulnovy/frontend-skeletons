# Pinnacle Portfolio

Pinnacle Portfolio is a standalone premium frontend skeleton for a personal creative portfolio. It includes a project gallery, expandable case study, services, testimonials, skills, contact form, and responsive mobile drawer.

## Files

- `index.html` contains the full HTML, CSS, and JavaScript implementation.
- `DESIGN_BOOK.md` documents the visual system, sections, interaction model, and customization guidance.
- `README.md` provides usage notes for the template.

## Usage

Open `index.html` in a browser or serve the folder with any static file server. The only external request is Google Fonts for Manrope and Inter.

## Features

- Responsive sticky navigation
- Keyboard-navigable mobile drawer
- Filterable project gallery
- Expandable case study details
- Testimonial carousel
- Skill bar visualization
- Accessible contact form shell
- Visible footer credit
