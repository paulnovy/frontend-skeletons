# Forge Startup Studio

Standalone premium frontend skeleton for a startup accelerator portal.

## Files

- `index.html` - standalone HTML with inline CSS and JavaScript
- `DESIGN_BOOK.md` - design tokens, layout guidance, accessibility notes, and content model
- `README.md` - usage and customization notes

## Features

- Sticky top navigation with required links and Apply Now CTA
- Keyboard-friendly mobile hamburger drawer
- Hero with accelerator positioning, stats strip, and live cohort command panel
- Program overview for Seed, Growth, and Scale tracks
- Portfolio showcase, application flow preview, events, mentor network, and footer
- Manrope headings and Inter UI loaded from Google Fonts
- Required cloud, ink, indigo, cyan, lime, and coral color system
- Semantic landmarks, visible focus states, Escape-to-close drawer behavior, and reduced-motion support

## Usage

Open `index.html` directly in a browser. The page has no runtime dependencies beyond the Google Fonts CDN and includes all styling and behavior inline.

## Customization

- Edit CSS custom properties in `:root` to tune spacing, radii, shadows, and opacity-based tints.
- Replace portfolio companies, mentor names, event dates, and program numbers with real accelerator data.
- Extend the three-step form preview into a connected application workflow.
- Swap the footer studio credit for the producing studio while keeping the visible credit line.
