# Orbit EdTech Campus

Standalone premium frontend skeleton for an online learning platform with course catalog browsing, progress tracking, interactive lessons, community discussion, and achievement badges.

## Files

- `index.html` contains the complete single-file interface with inline CSS and JavaScript.
- `DESIGN_BOOK.md` documents the concept, palette, typography, sections, accessibility, and responsive behavior.
- `README.md` records usage and verification guidance.

## Usage

Open `index.html` directly in a browser. The page has no build step and no runtime dependency beyond the Google Fonts CDN request for Manrope and Inter.

## Verification

Run the required HTML parser, line count, palette, font, and forbidden-text checks after creation. Responsive verification should be performed with Playwright at desktop `1440`, tablet `768`, and mobile `390`, including a mobile screenshot for visual inspection.
