# Wave Professional Network

## Category
Professional networking

## Description
Wave helps professionals find signal in their network with member-rich groups, structured mentorship, warm referral workflows, and job opportunities connected to trusted context.

## Features
- Standalone single-page HTML with inline CSS and JavaScript.
- Responsive layout with mobile hamburger navigation.
- CSS custom properties for fast theme editing.
- Realistic mock content tailored to professional networking.
- Hover, focus, active, transition, and animation states.
- CSS Grid and Flexbox structure for product-grade layouts.
