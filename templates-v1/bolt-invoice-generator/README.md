# Bolt Invoice Generator

An invoice generation workspace with template preview, editable line items, totals, payment status, and client-ready detail.

## Included

- Standalone `index.html` with inline CSS and JavaScript.
- Responsive mobile-first layout with a working hamburger navigation.
- Semantic HTML5 sections for hero, controls, cards, modules, workbench, people, flow, CTA, and footer.
- Google Fonts CDN only; no build step and no runtime dependencies.
- Required footer credit: "Built with Frontend Skeletons".

## Design Profile

- Category: Invoice tool
- Palette: White and green
- Primary accent: `#16a34a`
- Secondary accent: `#86c75a`

## Editing Notes

Theme tokens live in the `:root` block at the top of the inline stylesheet. Replace the realistic sample content with production data, connect the controls and card actions to application state, and keep the spacing scale intact for the intended premium feel.

## Suggested Use

Use this skeleton for product prototypes, sales demos, design explorations, or as a front-end starting point for a real application screen.
