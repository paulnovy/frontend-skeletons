# Beacon Waitlist

Beacon Waitlist is a standalone premium frontend skeleton for **Product waitlist page** experiences.

## Included

- `index.html` with inline CSS and JavaScript
- Responsive semantic HTML5 structure
- Mobile hamburger navigation
- Polished hover, focus, active, reveal, and counting states
- Realistic mock content tailored to product waitlist page
- Google Fonts CDN link only; no framework or runtime dependency

## Visual Direction

- Palette: dark gradient bg with electric blue glow
- Primary accent: `#38bdf8`
- Secondary accent: `#a78bfa`
- Font: Inter

## Usage

Open `index.html` directly in a browser or serve it from any static host. The file is self-contained apart from the Google Fonts request.
