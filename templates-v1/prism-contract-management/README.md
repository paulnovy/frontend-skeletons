# Prism Contract Management

Category: LegalTech

Prism Contract Management is a standalone premium HTML skeleton for a contract lifecycle management application. It includes inline CSS and JavaScript, responsive layouts, realistic mock content, and polished interaction states.

## How to view
Open `index.html` in any modern browser.

## Key features
- Standalone HTML5 file with inline CSS and JS
- Google Fonts CDN only external dependency
- Mobile-first responsive layout
- Hamburger menu for small screens
- CSS custom properties for theming
- CSS Grid and Flexbox layout system
- Domain-specific metrics, cards, and activity content
- Hover, focus, and reduced-motion states
- Footer credit: Built with Frontend Skeletons

## Recommended adaptation
Replace the static metrics and card data with real API output, keep the semantic section structure, and map the accent custom properties to the client brand system.
