# Matrix Comparison

Product comparison tool skeleton generated as a standalone premium HTML template.

## Contents

- `index.html` — semantic HTML5 with inline CSS and JavaScript.
- `DESIGN_BOOK.md` — design rationale, component notes, and adaptation guidance.
- `README.md` — usage and customization notes.

## Usage

Open `index.html` directly in a browser. The file has no runtime dependencies except the Google Fonts stylesheet declared in the document head.

## Included Behaviors

- Responsive navigation with a mobile hamburger menu.
- Polished hover, focus, active, and transition states.
- Interactive controls relevant to the skeleton, such as accordions, toggles, filters, tabs, or collapsible navigation.
- Realistic content structure tailored to: Product comparison tool.

## Customization

Adjust CSS custom properties in `:root` to re-theme quickly. Replace the mock copy and data blocks in the HTML with product, company, or customer-specific content.
