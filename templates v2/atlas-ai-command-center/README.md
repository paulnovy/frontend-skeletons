# Atlas AI Command Center

## Category
AI Operations Dashboard / Enterprise Command Center

## Description
Atlas AI Command Center is a premium standalone frontend skeleton for teams operating enterprise AI systems in production. It presents model health, workflow risk, governance posture, incident activity, cost pressure, logs, and AI-generated recommendations in one dense command center interface.

## How to View
Open `index.html` directly in a browser, or serve the folder with a local HTTP server:

```bash
cd "/home/marian/clawd/frontend-skeletons/templates v2/atlas-ai-command-center"
python3 -m http.server 3005
```

Then visit:

```text
http://localhost:3005/
```

## Key Features
- Standalone `index.html` with inline CSS and JavaScript.
- Google Fonts only: Space Grotesk, Inter, and IBM Plex Mono.
- Desktop three-column command center layout.
- Live operational summary with status chips, risk score, and next-best actions.
- Bento grid with healthy, warning, critical, and pending card states.
- Right rail with realistic AI recommendations.
- Tablet sticky status bar.
- Mobile hamburger menu, vertical priority cards, bottom navigation, and horizontal metrics.
- Accessible focus states, semantic landmarks, reduced-motion fallback, and keyboard-friendly drawer behavior.
- Footer credit: "Built with Frontend Skeletons".
