# Zen Wellness Retreat

A serene booking experience for restorative retreats, practitioner-led programs, quiet stays, and nature-centered recovery.

## Included

- Standalone `index.html` with inline CSS and JavaScript.
- Responsive mobile-first layout with hamburger navigation.
- Semantic sections for hero, overview, featured content, dashboard, people, timeline, CTA, and footer.
- Google Fonts CDN link only; no other external dependencies.

## Editing Notes

- Theme tokens live in the `:root` block at the top of the inline stylesheet.
- Navigation anchors are local page links and can be replaced with real routes.
- The visual frames are CSS-built content areas intended to be swapped for product screenshots, property imagery, charts, or project media.
- Footer credit is required: "Built with Frontend Skeletons".

## Suggested Use

Use this skeleton as a premium starting point for a production landing page or app-facing marketing surface. Replace mock content with real copy, connect forms to your backend, and keep the spacing scale intact for the intended polished feel.
