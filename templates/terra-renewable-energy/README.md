# Terra Renewable Energy

## Category
Renewable energy monitoring

## Description
A dark operations interface for renewable asset managers who need production, curtailment, storage, grid constraints, and weather-aware forecasts in one real-time control surface.

## Features
- Standalone single-page HTML with inline CSS and JavaScript.
- Responsive layout with mobile hamburger navigation.
- CSS custom properties for fast theme editing.
- Realistic mock content tailored to renewable energy monitoring.
- Hover, focus, active, transition, and animation states.
- CSS Grid and Flexbox structure for product-grade layouts.
- Footer credit: Built with Frontend Skeletons.
