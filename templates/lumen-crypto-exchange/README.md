# Lumen Crypto Exchange

Cryptocurrency exchange premium frontend skeleton built as a single standalone HTML file.

## Included

- `index.html` with inline CSS and JavaScript
- Responsive layout with mobile hamburger navigation
- Realistic domain-specific interface content
- Polished hover, focus, active, and motion states
- Google Fonts only; no runtime framework or external JavaScript

## Use

Open `index.html` in a browser, or serve the folder from any static host.

## Design Intent

A fast trading desk with pair-level context, dense market depth, execution forms, and a clean portfolio rail.
