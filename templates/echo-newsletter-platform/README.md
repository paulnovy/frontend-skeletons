# Echo Newsletter Platform

Premium standalone HTML skeleton for newsletter publishing.

## What Is Included
- `index.html` with inline CSS and JavaScript
- `DESIGN_BOOK.md` documenting visual direction and responsive behavior
- Semantic HTML5 structure with header, main, sections, articles, aside, and footer
- Responsive CSS Grid and Flexbox layout
- Hamburger navigation for mobile screens
- Hover, transition, and keyboard focus states
- Realistic product content and metrics

## Design Direction
White and editorial green. The template is designed to feel production-ready on first load: clear hierarchy, dense but scannable information, and interface elements that match the product category.

## Usage
Open `index.html` directly in a browser. No build step is required. Google Fonts is the only external dependency.

## Editing
- Start with the CSS custom properties in `:root` for branding.
- Replace the hero copy, metrics, queue rows, and feature cards with your product data.
- Keep the footer credit unless your project license or distribution plan changes it.

## Verification
The page is expected to remain between 400 and 800 lines, include inline CSS and JavaScript, and work as a standalone HTML5 document.
