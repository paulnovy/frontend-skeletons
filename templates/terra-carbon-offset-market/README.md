# Terra Carbon Offset Market

## Category
Carbon credit marketplace

## Description
Terra gives sustainability teams a marketplace view of carbon projects with registry status, vintage, co-benefits, retirement workflows, and audit-ready impact reporting.

## Features
- Standalone single-page HTML with inline CSS and JavaScript.
- Responsive layout with mobile hamburger navigation.
- CSS custom properties for fast theme editing.
- Realistic mock content tailored to carbon credit marketplace.
- Hover, focus, active, transition, and animation states.
- CSS Grid and Flexbox structure for product-grade layouts.
- Footer credit: Built with Frontend Skeletons.
